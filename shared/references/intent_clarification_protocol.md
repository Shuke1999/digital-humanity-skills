# Intent Clarification Protocol

## When to Clarify

Clarify intent before proceeding when:
1. The user's research domain is ambiguous between DH sub-fields (literary studies vs. history vs. linguistics)
2. The corpus or primary source is described vaguely (e.g. "old newspapers" — which country, century, collection?)
3. The methodology orientation is unclear (computational? interpretive? both?)
4. The output type is unspecified (journal article? thesis chapter? conference abstract?)
5. The user level is uncertain (undergraduate? graduate? faculty?) — this affects depth and scaffolding

## How to Clarify

Ask **at most 2–3 targeted questions**. Never ask more than 3 at once. Frame questions to surface information needed for the next step, not to gather every possible variable.

Good clarification question format:
> "To help you best — are you working with [Option A] or [Option B]? And roughly what career stage? (This helps me calibrate depth and scaffolding.)"

Avoid:
- Open-ended "Tell me about your project" without a specific focus
- Asking the same question twice in different words
- Asking about things you can infer (e.g. if a user mentions "my dissertation," infer graduate level)

## Disambiguation Table

| Ambiguity | Ask About |
|---|---|
| Domain unclear | Sub-field: literary, historical, linguistic, cultural, archival |
| Time period vague | Century + geography of source materials |
| Corpus undefined | Digitized vs. born-digital; size estimate; known collection |
| Method unspecified | How much computational work vs. traditional archival/interpretive |
| Output unclear | Paper type, target venue, word count range |
| User level | Career stage and prior DH experience |

## [direct-mode] Prefix

If the user begins their message with `[direct-mode]`, skip all clarification and proceed directly with the most reasonable interpretation. This is a power-user signal.

## Anti-Patterns

- **Do not** ask for information the user has already provided
- **Do not** create false choices (offering only computational vs. only interpretive as if hybrid is impossible)
- **Do not** delay the session with lengthy explanations before asking the clarifying question
